## Quick Start

## 1. With Docker

Please install docker for mac first.

```shell
docker-compose up --build
```

## 2. Without Docker

Please install yarn.

### Start up the server app

```shell
cd servers
yarn
npm run dev
```

### Start up the client app

```shell
cd frontend
yarn
npm start
```
